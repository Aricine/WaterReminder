# 💧 喝水提醒小助手

一个常驻系统托盘的喝水提醒程序，每隔一段时间自动弹出 Windows 通知提醒你喝水。

## 功能

- 🔔 定时弹出系统通知提醒喝水
- ⏱ 自定义提醒间隔（30/45/60/120 分钟）
- ⏸ 可随时暂停/恢复提醒
- 💧 支持「立即提醒一次」手动触发
- 🎨 系统托盘水滴图标（启用时蓝色，暂停时灰色）

## 使用方式

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动

双击 `start.bat` 或在终端运行：

```bash
python water_reminder.py
```

### 3. 托盘操作

启动后程序图标出现在系统托盘（任务栏右下角），右键点击图标：

| 菜单项 | 说明 |
|--------|------|
| ✅ 已启用 / ❌ 已暂停 | 切换提醒开关 |
| 💧 立即提醒一次 | 立刻弹出喝水通知 |
| ⏱ 提醒间隔 → | 选择下次提醒间隔 |
| 🚪 退出 | 关闭程序 |

## 开机自启动（已配置）

✅ **已设置开机自启动！** 每次开机后喝水提醒会自动运行在系统托盘。

设置原理：
1. `setup_startup.py` 创建一个 VBS 静默脚本 (`start_silent.vbs`)
2. 使用 `pythonw.exe` 无窗口运行主程序
3. 在 `启动` 文件夹添加快捷方式

> 如果之后想移除开机自启动，只需删除 `启动` 文件夹里的 `喝水提醒.lnk`

## 配置

配置文件 `config.json` 自动保存在程序同目录下，可手动编辑：

```json
{
  "interval_minutes": 30,
  "enabled": true
}
```
